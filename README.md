# Chillify

A Flutter music app made with Provider and BLoC pattern. (Works on Android for now, iOS coming soon)

UI heavily inspired by: https://dribbble.com/shots/6523216-Music-Player-Application/attachments

## Current Features
[✓] Retrieve all songs

[✓] Pause

[✓] Play

[✓] Seek

[✓] Repeat

[✓] Shuffle

[✓] Now Playing

[✓] add to favorites

[✓] Albums

[✓] Well-designed UI
